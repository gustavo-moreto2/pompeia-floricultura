# Pompeia Floricultura

Site institucional e vitrine digital da Pompeia Floricultura, desenvolvido com Next.js, React, Tailwind CSS e integração com Supabase.

## Sobre o projeto

O projeto reúne páginas públicas da floricultura, galeria, produtos, serviços, avaliações, contato e área administrativa para gerenciamento de informações do site.

## Tecnologias

- Next.js 15
- React 19
- TypeScript
- Tailwind CSS
- Supabase
- ESLint

## Como rodar localmente

Instale as dependências:

```bash
npm install
```

Crie um arquivo `.env.local` com base no `.env.example` e configure as variáveis necessárias.

Inicie o servidor de desenvolvimento:

```bash
npm run dev
```

Acesse:

```bash
http://localhost:3000
```

## Scripts disponíveis

```bash
npm run dev
npm run build
npm run start
npm run lint
```

## Validação

Antes de publicar, rode:

```bash
npm run lint
npm run build
```

## Publicação

O projeto pode ser publicado na Vercel ou em qualquer hospedagem compatível com aplicações Next.js.
